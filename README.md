Sports Score Predictor
A simple Python project to predict sports match outcomes based on recent win/loss history using basic logic.

About
This project demonstrates a foundational approach to predicting sports scores by comparing recent match performances. Ideal as a programming beginner's guide for logic-based prediction.

Features
Predicts the winner or draw for a match between two teams

Easy to modify for more teams or other sports
